## 들여쓰기
4개의 공백 또는 하나의 탭으로 들여쓰기
들여쓰지 않은 행을 만날 때 까지 변환이 계속됨.

## 코드
```py
e_tot = 0
o_tot = 0
for x in range(1, 101):
    if x % 2 == 0  :
        e_tot += x
    else :
        o_tot += x

print('1 ~ 100 까지 짝수 합 : {0}, 홀수 합 : {1}'.format(e_tot, o_tot))
```

' ``` ' 으로 묶거나

```
<pre><code>{code}</code></pre>
```
으로 처리함

첫 '```' 뒤에 js, py 등 과같은 사용하는 언어를 선언하면 문법 강조 가능
